FORENSICORE is an advanced AI-powered digital forensic and behavioral risk analysis platform designed to enhance cybersecurity, forensic investigation, and insider threat detection.
The system uses machine learning models such as Random Forest (for risk prediction) and Isolation Forest (for anomaly detection) to analyze user behavior based on key indicators like login frequency, file access patterns, session duration, failed attempts, and suspicious activities.
It not only predicts risk probability but also performs behavioral baseline comparison, detects deviations, and provides explainable AI insights to justify decisions.
The platform generates digitally signed forensic reports with:
Evidence integrity verification (SHA256 hashing)
Audit logs and traceability
AI-based risk analysis
Visual graphs (Risk Trend, Baseline Analysis, ROC Curve, Confusion Matrix)
Additionally, it supports:
Secure evidence storage (ZIP locking)
Camera-based scan logging (optional)
Real-time risk scoring
Predictive trend analysis
